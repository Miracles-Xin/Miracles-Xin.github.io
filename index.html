第五章 列表与元组

5.1 序列概述

​ 序列是一块用于存放多个值的连续内存空间，并且按一定顺序排列，每个值（称为元素）都分配一个数字，称为索引或位置。

​ 在Python中，序列结构主要有列表、元组、集合、字典和字符串；对于这些序列结构有以下几个通用的操作，其中，集合和字典不支持索引、切片、相加和相乘操作。

5.1.1 索引

​ 序列中的每一个元素都有一个编号，也成为索引（Indexing）；通过索引可以访问序列中的任何元素

​ 索引从0开始递增；Python的索引也可以是负数，从-1开始，从右向左计数，即从最后一个元素开始计数。

5.1.2 切片

sname[start:end:step]

​ 切片（Slicing）操作是访问序列中元素的另一种方法，它可以访问一定范围内的元素；

​ 在进行切片操作时，如果指定了步长，那么将按照该步长遍历序列的元素，否则将一个个遍历序列；

​ 如果想要复制整个序列，可以将start和end参数都省略，但是中间的冒号需要保留，例如：verse[:]

5.1.3 序列相加

verse1 + verse2

5.1.4 乘法

verse * 3:即将原序列复制三遍构成一个新的序列

emptylist = [None] * 5:将创建一个长度为5的列表，每个元素都是None，表示什么都没有

5.1.5 检查某个元素是否是序列的成员

5.3.5 元组和列表的区别

  1. 列表属于可变序列，其元素可以随时修改和删除；元组属于不可变序列，除非整体替换，其中元素不可以修改
  2. 列表可以使用append()、extend()、insert()、remove()和pop()等方法实现添加和修改元素，而元组不能
  3. 可以使用切片访问和修改列表中的元素，而元组只支持访问，不支持修改
  4. 元组比列表访问和处理速度更快
  5. 列表不能作为字典的键，而元组可以

6.1 字典

​ 字典（dictionary）和列表类似，也是可变序列，不过与列表不同的是，它是无序的可变序列，保存的内容是以“键-值对”的形式存放

6.1.1 字典的创建和删除

dictionary = {'key1':'value1','key2':'value2',...,'key n':'value n',}

​ 创建空字典：dictionary = {}或者dictionary = dict()

  1. 通过映射函数创建字典

    dictionary = dict(zip(list1,list2))

    zip()函数：用于将多个列表或元组对应位置的元素组合成元组，并返回包含这些内容的zip对象；如果想得到元组，可以将zip对象 使用tuple()函数转换为元组，列表使用list()转换。

    list1：表示一个列表，用于指定要生成字典的键

    list2：用于指定要生成字典的值

  2. 通过给定的“键-值对”创建字典

    dictionary = dict(key1=value1,key2=value2,...,key n=value n)

    在Python中，还可以使用dict对象的fromkeys()方法创建值为空的字典：

    dictionary = dict.fromkeys(list1)

    通过已有的元组和列表创建字典：

    dict1 = {tuple:list}

  3. 删除字典

    del dictionary

    dictionary.clear()

6.1.2 访问字典

  1. print函数直接打印字典，也可以通过键输出对应的值

  2. dictionary.get(key[,default])

    defalut：为可选项，用于当指定的键不存在时，返回一个默认值，如果省略则返回None

第七章 字符串

7.1 字符串编码转换

​ 在Python中，有两种常用的字符串类型，分别为str和bytes。其中，str表示Unicode字符（ASCII或者其它）；bytes表示二进制数据（包括编码的文本）。这两种类型的字符不能拼接在一起使用。通常情况下，str在内存中以Unicode表示，一个字符对应若干个字节，但是如果在网络上传输，或者保存到磁盘上，就需要把str转换为bytes类型。

​ str和bytes之间可以通过encode()和decode()方法进行转换。

7.1.1 使用encode（）方法编码

​ str.encode([encoding="utf-8"][,errors="strict"])

​ str表示要转换的字符串

​ encoding=”utf-8”：可选参数，用于指定进行转码时采用的字符编码，默认为UTF-8，简体中文为gb2312

​ errors=”strict”：可选参数，指定错误处理方式，strict（遇到非法字符抛出异常）、ignore（忽略）、replace（用“？”替换非法字符）、xmlcharrefreplace（使用XML的字符引用），默认为strict

7.1.2 使用decode（）方法解码

​ bytes.decode([encoding="utf-8"][,errors="strict"])

7.2 字符串常用操作

7.2.1 拼接字符串

​ 使用“+”运算符可完成对多个字符的拼接

​ 字符串不允许直接与其它类型的数据拼接，可以利用str()函数转换成字符串

7.2.2 计算字符串的长度

​ len(string)

​ len()函数不区分英文、数字和汉字，所有字符都认为是一个

​ 获取字符串实际所占的字节数：可以通过encode()方法进行编码后再进行获取

7.2.3 截取字符串

​ string[start:end:step]

7.2.4 分割、合并字符串

  1. 分割字符串

    str.split(sep,maxsplit)

    str：字符串名字

    sep：用于指定分隔符，可以包含多个字符，默认为None

    maxsplit：可选参数，用于指定分割的次数

  2. 合并字符串

    strnew = string.join(iterable)

    string：字符串类型，用于指定合并时的分隔符

    iterable：可迭代对象，该对象以字符串表示，包括很多元素

7.2.5 检索字符串

  1. count()方法：输出个数

    str.count(sub[,start[,end]])

    sub：要检索的字符串

  2. find()方法：输出第一次出现的位置索引

    str.find(sub[,start[,end]])

  3. index()方法

    str.index(sub[,start[,end]])

    当指定的字符串不存在时报错

  4. startswith()方法

    str.startswith(prefix[,start[,end]])

    检索字符串是否以指定字符串开头

  5. endswith()方法

    str.endswith(suffix[,start[,end]])

7.2.6 字母的大小写转换

  1. lower()方法

    str.lower()

  2. upper()方法

    str.upper()

7.2.7 去除字符串中的空格和特殊字符

  1. strip()方法

    用于去掉字符串左右两侧的空格和特殊字符

    str.strip([chars])

    chars用于指定要去除的字符，可以指定多个

  2. lstrip()方法

    用于去掉字符串左侧的空格和特殊字符

    str.ltrip([chars])

  3. rstrip()方法

    用于去掉字符串左侧的空格和特殊字符

    str.rtrip([chars])

7.2.8 格式化字符串

  1. 使用%操作符

    '%[-][+][0][m][.n]格式化字符'%exp

    -：可选参数，用于指定左对齐，正数前方无符号，负数前方加负号

    +：可选参数，用于指定右对齐，正数前方加正号，负数前方加负号

    0：可选参数，表示右对齐，正数前方无符号，负数前方加负号，用0填充空白处

    m：可选参数，表示占有宽度

    .n：可选参数，表示小数点后保留的位数

    格式化字符：用于指定类型，比如%s、%c等等

    exp：表示要转换的对象

  2. 使用字符串对象的format()方法

    str.format(args)

    str用于指定模板

    args用于指定要转换的项，多项用逗号分隔

    创建模板：

    ​ {[index][:[fill]align][sign][#][width][.precision][type]}

    ​ fill：可选参数，指定空白处填充的字符

    ​ align：可选参数，用于指定对齐方式，<左对齐;>右对齐;=右对齐符号放在填充内容的最左侧;^内容居中配合width使用。

    ​ sign：可选参数，用于指定有无符号数

    ​ #：可选参数，对于进制数显示前缀

    ​ width：可选参数，指定所占宽度

    ​ .precision：可选参数，指定保留的小数位数

    ​ type：可选参数，指定类型；例如：S（对字符串类型格式化）、b（将十进制转换为二进制再格式化）、D、o、C、x、e、f、g、%
